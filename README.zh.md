## 🔄 开发项目时的标准流程

### 用户说「我要开发 X 项目」之后：

```
1. ✅ 分析项目类型
   ↓
2. ✅ 检查环境配置
   → 去 Global-Dev-Setup 找对应的环境模板
   → 检查环境是否已配置
   → 如果没配置，立即配置
   ↓
3. ✅ 检查需要的开发技能
   → 去 AI-SKILL 找相关技能
   → 如果没加载，立即加载
   ↓
4. ✅ 检查需要的工具
   → 去 Mcp-Market 找需要的工具
   → 如果没配置，立即配置
   ↓
5. ✅ 检查需要的提示词
   → 去 PromptHub 找
   ↓
6. ✅ 检查需要的 API
   → 去 API-Market 找
   ↓
7. ✅ 开始开发！
   ↓
8. ✅ 全程记录项目上下文
```
